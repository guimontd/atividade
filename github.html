JavaScript

1. JavaScript e Java são a mesma coisa?

Não, JavaScript e Java são linguagens completamente diferentes. Java é uma linguagem de programação orientada a objetos, desenvolvida para ser executada em qualquer plataforma com a JVM (Java Virtual Machine). Já o JavaScript é uma linguagem de script que roda no navegador e foi criada para adicionar interatividade aos sites.

2. Quando surgiu o JavaScript?

O JavaScript foi criado em 1995 e inicialmente era chamado de Mocha. Foi renomeado para LiveScript e, mais tarde, para JavaScript, devido a uma parceria entre a Netscape e a Sun Microsystems.

3. Quem foi o responsável pela criação do JavaScript?

O JavaScript foi criado por Brendan Eich, um programador da Netscape Communications Corporation, que desenvolveu a linguagem em apenas 10 dias.

4. Qual o caminho trilhado pelo responsável da criação do JavaScript? Trata-se de alguma empresa ainda no mercado?

Brendan Eich trabalhou na Netscape, onde criou o JavaScript. Posteriormente, ele cofundou a Mozilla Foundation, responsável pelo navegador Firefox. Atualmente, Brendan Eich é CEO da Brave Software, empresa que desenvolve o navegador Brave, voltado para privacidade e segurança.
